# 요구사항 화면 대응 현황

## 목적

요구사항 PDF에 포함된 기능 중 현재 PoC 화면으로 대응된 영역과 미대응 영역을 정리한다.

## 주요 영역

- **메인 포털**: `Portal Dashboard`에서 Hero, 통합검색, 요약 카드, 공지/자산/프로젝트/반출/크레딧/K-RMF/시스템 상태/바로가기 제공
- **로그인/로그아웃**: Backstage `SignInPage`에서 Guest/Keycloak 선택, Keycloak OIDC 연동 완료
- **사용자 관리**: `Admin Console > 사용자 관리`에서 사용자 목록, 상태, 접근 서비스 등을 조회
- **사용자 등록 승인**: `Admin Console > 사용자 등록 신청`에서 본인확인 상태, 승인/반려 UI 제공
- **권한 관리**: `Admin Console > 권한 / 역할 관리`에서 역할별 메뉴/데이터/프로젝트/승인 권한 표시
- **로그인 정책 관리**: `Admin Console > 로그인 정책`에서 허용 기간, 실패 횟수, 중복 로그인, 해제 방식 표시
- **비밀번호 정책 관리**: `Admin Console > 비밀번호 정책`에서 복잡도, 변경 주기, 만료 잠금 방식, 재사용 제한 표시
- **이력/감사로그 관리**: `Admin Console > 감사 로그`에서 로그인/비밀번호/권한/반출/검색/정책 변경 이력 표시
- **메뉴/프로그램 관리**: `Admin Console > 메뉴 관리`에서 트리 구조 메뉴, 경로, 기능 연결, 접근 역할 표시
- **서비스 카탈로그**: 아직 화면 미대응
- **제안 게시판**: 아직 화면 미대응
- **통합검색**: `Integrated Search`에서 OpenSearch `portal-catalog` 연동 검색, 필터/상세 제공
- **반출/반입 관리**: `Export Approval`에서 신청/상세/등록, 상태/보안등급/감사로그 표시
- **프로젝트/워크스페이스 관리**: `Project Workspace`에서 목록/상세/생성 신청, 참여자/리소스 표시
- **크레딧 관리**: `Credit Manager`에서 잔여/사용/충전 내역, 경고, 자원 단가 표시

## 후속 단계

- **14단계**: 사용자 등록/승인 화면 상세화
- **15단계**: 권한/역할/메뉴 관리 상세화
- **16단계**: 로그인/비밀번호 정책 및 감사로그 상세화
- **17단계**: 서비스 카탈로그/제안 게시판 화면 추가
- **18단계**: 요구사항 추적표 최종 정리
