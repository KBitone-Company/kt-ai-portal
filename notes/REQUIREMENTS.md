# REQUIREMENTS

## 프로젝트 목표

MacBook 개발환경에서 VS Code로 개발 가능한 `KT AI/Data Platform Portal PoC`를 단계적으로 구축한다.

## 기술 기준

- 포털 프레임워크: Backstage
- 인증/권한: Keycloak
- 데이터/AI 카탈로그: OpenMetadata
- 검색엔진: OpenSearch
- 업무 특화 기능: Backstage 자체 플러그인 개발

## 개발환경

- OS: macOS
- IDE: VS Code
- Container: Docker Desktop
- Runtime: Node.js, Yarn

## 개발방식

- 단계별 구현
- 각 단계 완료 후 실행/검증 방법 명시
- PoC 수준이나 운영 확장 가능한 구조 설계

## 주요 산출물

1. 실행 가능한 Backstage 기반 포털 PoC
2. Keycloak 인증 연동 구조
3. OpenSearch 검색엔진 구조
4. OpenMetadata 카탈로그 연동 설계
5. 프로젝트/워크스페이스 관리 플러그인
6. 반출/반입 관리 플러그인
7. 크레딧 관리 플러그인
8. K-RMF 증빙관리 플러그인
9. 통합검색 화면
10. 전체 아키텍처 문서
11. WBS 및 개발완료 현황표
12. 변경 이력 요약 문서
